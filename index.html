RentSmart

RentSmart is a full-stack CRUD web application demo showing (hypothetical) reviews for landlords in the Denver area. A partner and I created it together while in our General Assembly web development immersive using Node.js, Express, MongoDB, and Bootstrap.

The application features a search function supporting partial matches for landlord names, encrypted login, and cookies for persistent sessions.
